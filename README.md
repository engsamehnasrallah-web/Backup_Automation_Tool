# Backup Automation Tool

## Overview

A Python-based automation tool that creates compressed backups of folders.

---

## Features

- Folder Backup
- ZIO Compression
- Timestamp-based Naming
- Automatic Backup Directory
- Backup Logs

---

## Technologies

- Python
- shutil
- datetime
- os

---

### Usage

~~~bash
python backup.py
~~~
